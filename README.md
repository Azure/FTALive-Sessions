# FastTrack for Azure Live - Session Handouts 

Hello! This repository has been created to share our [FastTrack for Azure Live](https://fasttrack.azure.com/live) session materials in one easy to manage location. It is a dynamic and regularly updated repository so please bookmark and keep an eye on it.

Note that the information contained is publically available and as such is maintained by Azure Engineers regardless of timezone.

- ### [View Upcoming FTA Live Schedules &rarr;](https://fasttrack.azure.com/live)
